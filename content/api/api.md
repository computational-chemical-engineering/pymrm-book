# Module Overview

An overview of all PyMRM modules and their public API.

| Module | Description |
|--------|-------------|
| [pymrm.convect](generated/overview/pymrm.convect) | Convective flux schemes (upwind, TVD, MUSCL, etc.) |
| [pymrm.coupling](generated/overview/pymrm.coupling) | Interface coupling and index translation utilities |
| [pymrm.grid](generated/overview/pymrm.grid) | Grid generation functions |
| [pymrm.helpers](generated/overview/pymrm.helpers) | Coefficient matrix construction helpers |
| [pymrm.interpolate](generated/overview/pymrm.interpolate) | Interpolation between staggered and cell-centre grids |
| [pymrm.numjac](generated/overview/pymrm.numjac) | Numerical Jacobian computation |
| [pymrm.operators](generated/overview/pymrm.operators) | Differential operator matrices (gradient, divergence) |
| [pymrm.solve](generated/overview/pymrm.solve) | Nonlinear solver utilities |
